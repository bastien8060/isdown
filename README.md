# isdown
Bash script checks if a website or Ip address is down or not.


<h1>Installation:</h1>
<ul>
<li>
<h2>Arch Linux</h2>
</li>
<p>
<pre>sudo make install</pre>
</p>


<li>
<h2>Other Distros: (soon to come)</h2>
</li>

</ul>
